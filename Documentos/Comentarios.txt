Servicios web SOAP
------------------
WsSoapGlashFish\FechaWsMvn:
	uso de métodos One Way. Uso de Message handlers
WsSoapGlashFish\FactorialWsMvn:
	uso de EJBs para crear servicios web. Algunas características de JSE8 de cara al diseño
SoapAsync\EjemploSoapAsyncMvn: servicio web concurrente
SoapAsync\ClienteSoapAsyncMvn: cliente concurrente
SoapAsync\ClienteSoapAsyncJSE7: cliente concurrente con Java 7
WsSoapGlashFish\ClienteAjaxFactorialWsMvn:
	Llamar a un servicio Web empleando JavaScript

Servicios web RESTful estándar Java
------------------------------------
RestJEE\FraseWsMaven:
	Uso de JSON para intercambio de mensajes.
	Uso de "traductores" de objetos al formato JSON
	Instrucciones para ver el archivo "wadl" generado automáticamente por GlassFish
RestJEE\FraseWsClientMaven:
	Cliente simple del programa FraseWsMaven
RestJEE\FraseWsAsyncMaven:
	El servicio web es asíncrono.
	Traducción de mensajes a JSON sin "traductores"
RestJEE\ClienteFraseAsyncMaven:
	Cliente simple del programa FraseWsAsyncMaven
RestJEE\EjemploRestServletMaven:
	Implementación de un RESTful con una Servlet
RestJEE\ClienteJavaScriptRestServletMaven:
	Acceso a un RESTful desde una página Web usando AJAX y JSON
EjemploRestSoap:
	Integración con servicios SOAP
RestJEE\EjemploRestSoapClient: 
	Cliente RESt para EjemploRestSoap
RestJEE\EjemploRestXml:
	Servicio web REStful que intercambia objetos en formato XML
RestJEE\EjemploRestXmlClient:
	Cliente para probar EjemploRestXml
